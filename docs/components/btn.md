# Btn

## Usage

<VBtn>Button</VBtn>

```vue
<v-btn>Button</v-btn>
```

## Props

| Prop         | Type    | Default | Description                          |
| ------------ | ------- | ------- | ------------------------------------ |
| **disabled** | boolean | false   | Disable the ability to use component |
| **rounded**  | boolean | false   | Set border radius to 5px             |
