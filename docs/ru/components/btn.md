## Props

| Prop         | Type    | Default | Description                                    |
| ------------ | ------- | ------- | ---------------------------------------------- |
| **disabled** | boolean | false   | Отключить возможность использования компонента |
| **rounded**  | boolean | false   | Добавить скругление углов в 5px                |
